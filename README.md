# Quotes to Scrape - Web Scraping Practice

This is a beginner web scraping practice project built using Python.

The notebook:

- Downloads quote pages from https://quotes.toscrape.com/
- Saves each page as an HTML file
- Parses the saved HTML using BeautifulSoup
- Extracts quotes that have the **"life"** tag
- Stores the author and quote in a Pandas DataFrame

## Technologies Used

- Python
- Requests
- BeautifulSoup
- Pandas
